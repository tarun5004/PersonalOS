# ADR-0009: Use MIT License

## Context

Personal OS is intended to be open-source, self-hostable, customizable, and community-driven.

The license should be permissive and familiar to contributors.

## Decision

Use the MIT License for V1.

## Alternatives considered

- Apache-2.0
- GPL-3.0
- AGPL-3.0
- No license until release

## Consequences

MIT makes the project easy to use, fork, modify, and redistribute.

The root `LICENSE` file will be created during Phase 1, not Phase 0-C.

## Status

Accepted for V1.
